### Step 2: Edit with CapCut (free)
Import recordings, add transitions, captions, and music.

## CapCut / Canva Editing Tips

1. **Auto-captions**: Enable in CapCut for accessibility
2. **Transitions**: Use simple fade or slide (not flashy)
3. **Music**: Use royalty-free lo-fi or tech background music (CapCut has free library)
4. **Speed**: Speed up loading/transition moments to 2x
5. **Text overlays**: Add feature names as text when showing each screen
6. **Resolution**: Export at 1080p (1920x1080)
7. **Duration**: Aim for 2:00 - 3:00 max

## Alternative: Canva AI Video

If you want a more polished look without screen recording:

1. Go to canva.com -> Create a Video
2. Use "Tech Product Demo" template
3. Add screenshots of each app screen
4. Use Canva's text-to-speech or record your own voiceover
5. Add animated transitions between screens
6. Export as MP4
